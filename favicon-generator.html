EduGuide Favicon Generator

Instructions to generate PNG favicon:

  1. Right-click on any of the favicon displays below
  2. Select "Save image as..." or "Copy image"
  3. Save as "favicon.png" in your public folder
  4. For Apple touch icon, use the 180x180 version
  5. Delete this file after generating the favicon
